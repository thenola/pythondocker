# Методы установки Python

PythonDocker поддерживает несколько методов установки версий Python:

## 1. Автоматическая установка через MSI (Windows)

Для Windows программа может автоматически скачать и установить Python через MSI установщик:

- **Python 2.7**: Скачивает MSI установщик с python.org и устанавливает его автоматически
- **Python 3.x**: Скачивает embeddable версию (ZIP архив) и распаковывает её

### Требования:
- Windows с правами администратора (для MSI установки)
- Доступ к интернету

## 2. Использование pyenv

Если у вас установлен pyenv, программа автоматически использует его для установки и управления версиями Python.

### Установка pyenv:

**Windows:**
```bash
# Через pyenv-win
git clone https://github.com/pyenv-win/pyenv-win.git %USERPROFILE%\.pyenv
```

**Linux/macOS:**
```bash
curl https://pyenv.run | bash
```

### Преимущества pyenv:
- Легкое управление несколькими версиями Python
- Автоматическая компиляция из исходников
- Поддержка всех версий Python

## 3. Системный Python

Программа автоматически ищет установленные версии Python в системе:

- **Windows**: Через Python Launcher (`py -2.7`) или в PATH
- **Linux/macOS**: Через команды `python2.7`, `python3.11` и т.д.

## Порядок попыток установки

Программа пытается установить Python в следующем порядке:

1. ✅ Проверка уже установленной версии в `~/.pythondocker/python/`
2. ✅ Поиск системного Python нужной версии
3. ✅ Установка через pyenv (если доступен)
4. ✅ Автоматическая установка через MSI/embeddable (Windows)
5. ✅ Скачивание исходников (Linux/macOS, требует компиляции)

## Рекомендации

- **Для Windows**: Рекомендуется использовать pyenv-win или установить Python вручную
- **Для Linux/macOS**: Рекомендуется использовать pyenv для простоты управления версиями
- **Для быстрого старта**: Установите нужные версии Python вручную или через pyenv

## Примеры

```bash
# Программа автоматически попробует все методы
pythondocker --python 2.7 script.py

# Если Python не найден, вы получите инструкции по установке
```
