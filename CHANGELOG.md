# История изменений

## Версия 2.0.0 - Собственная система контейнеризации

### Основные изменения

- ✅ **Удалена зависимость от Docker**: Программа теперь работает полностью без Docker
- ✅ **Собственная система изоляции**: Использует виртуальные окружения Python вместо Docker контейнеров
- ✅ **Автоматическое скачивание Python**: Скачивает нужные версии Python с python.org
- ✅ **Управление версиями**: Автоматически устанавливает и управляет различными версиями Python
- ✅ **Кроссплатформенность**: Работает на Windows, Linux и macOS без дополнительных зависимостей

### Новые модули

- `python_installer.py` - скачивание и установка версий Python
- `environment_manager.py` - управление изолированными виртуальными окружениями

### Удаленные модули

- `docker_manager.py` - больше не нужен, заменен на `environment_manager.py`

### Технические детали

- Версии Python хранятся в `~/.pythondocker/python/`
- Виртуальные окружения создаются в `~/.pythondocker/envs/`
- Использует стандартную библиотеку Python (urllib, subprocess, pathlib)
- Для Python 2 создает простое окружение без virtualenv
- Для Python 3 использует встроенный модуль venv

### Преимущества

1. **Не требует Docker**: Работает на любой системе с Python 3.7+
2. **Легковесность**: Не нужно устанавливать и запускать Docker
3. **Быстрый старт**: Не нужно ждать загрузки Docker образов
4. **Простота**: Использует только стандартную библиотеку Python
5. **Гибкость**: Легко расширять и настраивать

### Зависимости

- Удалена зависимость: `docker>=6.0.0`
- Теперь использует только стандартную библиотеку Python 3.7+
